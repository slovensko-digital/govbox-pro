# Správa

Správa v elektronickej schránke je komunikáciou s orgánom verejnej moci a spravidla nahrádza listinnú podobu.

## Atribúty správy

- **Názov** - identifikácia správy
- **Odosielateľ** - orgán verejnej moci
- **Predmet** - stručný popis obsahu
- **Adresát** - príjemca správy
- **Obsah** - samotný text správy
- **Prílohy** - dokumenty pripojené k správe
