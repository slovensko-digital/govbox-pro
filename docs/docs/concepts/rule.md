# Pravidlo

Pravidlo je skupina podmienok a akcií, ktoré sa majú vykonať pri nejakej udalosti v schránke.

## Využitie pravidiel

Pravidlá sa dajú použiť na základnú automatizáciu procesov spracovania správ alebo vlákien.

## Štruktúra pravidla

| Časť | Popis |
|------|-------|
| **Udalosť** | Spúšťač pravidla (napr. nová správa) |
| **Podmienky** | Kritériá, ktoré musí udalosť spĺňať |
| **Akcie** | Činnosti, ktoré sa vykonajú pri splnení podmienok |

## Príklad použitia

**Automatické označovanie správ:**
Vytvorenie štítka "Financie" s pravidlom označovania pre správy prichádzajúce od:
- Poisťovní
- Úradu práce, sociálnych vecí a rodiny
- Daňového úradu
