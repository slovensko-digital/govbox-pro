# Vlákno

Vlákno predstavuje súvisiace správy spájané dokopy a zobrazuje chronologickú históriu komunikácie s úradom.

## Vlastnosti vlákien

- Obsahujú jednu alebo viacero správ
- Zobrazujú chronologické zobrazenie komunikácie
- Môžu byť označené štítkami
- Je možné ich premenovať
