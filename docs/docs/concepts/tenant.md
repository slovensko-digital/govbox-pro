# Tenant

Tenantom sa rozumie skupina schránok, používateľov a pravidiel, ktorá riadi interné prístupy a nastavenia.

## Charakteristiky tenanta

- Zjednocuje správu viacerých schránok
- Umožňuje centralizovanú správu používateľov
- Definuje pravidlá pre automatizáciu
- Riadi prístupové oprávnenia
