# Používateľ

Používateľ je osoba využívajúca službu GovBox PRO.

## Charakteristiky používateľa

- Každá osoba sa do schránky prihlasuje vlastnými prihlasovacími údajmi
- Administrátori môžu pridávať do tenantov ďalších používateľov
- Používateľom je možné udeliť plný alebo čiastočný prístup k správam

## Prístup k správam

Prístup k správam je možné ovplyvniť pomocou:
- Štítkov
- Skupín
